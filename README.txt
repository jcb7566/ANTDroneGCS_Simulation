AntDroneGCS Simulation README
================================

These files use image data captured by two cooperative drones and run the same algorithms
implemented in C++.
______________________________________
Software Packages:

- OpenCV 2.4.5
- mexopencv repo (OpenCV wrap-around functions)
- Windows SDK (or another compatible compiler, see link)
- MATLAB R2012a

Follow the instructions at: http://www.cs.stonybrook.edu/~kyamagu/mexopencv/
to install OpenCV and the mexopencv repo

*note: if you removed the C++ redistributables to install Windows SDK, repair the programs
before restarting your computer. You might have deleted the C++ runtime *
